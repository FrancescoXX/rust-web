# Security Policy

## Supported versions

This is an early-stage project. Security updates are currently applied to the
latest commit on the `dioxus` branch.

## Reporting a vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, report privately to:

- `maintainers@rust-web-tcg.local` (replace with your real project email)

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

We will acknowledge receipt within 72 hours and provide a follow-up status as
soon as we validate the report.

## Disclosure

After a fix is available, maintainers may publish a security advisory and
credit the reporter (if desired).
