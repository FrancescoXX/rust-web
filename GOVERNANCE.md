# Governance

## Project status

`rust-web-tcg` is a community-driven, maintainers-led open-source project.

## Decision-making

- Small changes: handled in pull requests with maintainer review.
- Larger changes (architecture, major design direction): discussed in issues
  before implementation.
- In case of disagreement, maintainers seek rough consensus and make the final
  call to keep the project moving.

## Releases

- Releases are cut by maintainers when a meaningful set of changes is ready.
- Versioning follows SemVer where practical.
- Notable changes are documented in `CHANGELOG.md`.

## Becoming a maintainer

Frequent, high-quality contributors may be invited to become maintainers based
on sustained participation, constructive collaboration, and project fit.
